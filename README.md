\# AI-Based DTM Generation \& Drainage Network Design



This project uses LiDAR data and machine learning to generate Digital Terrain Models (DTM) and extract intelligent drainage networks.



\## Features

\- LiDAR point cloud processing

\- Random Forest classification

\- DTM generation

\- Flow direction \& accumulation

\- Drainage network extraction

\- 3D visualization
